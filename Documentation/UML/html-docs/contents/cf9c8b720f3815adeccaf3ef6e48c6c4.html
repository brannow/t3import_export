t3import_export

Description

A versatile import and export extension for the TYPO3 CMS. Exports and imports from different sources to different targets. (XML, CSV, MySQL-Database). Several pre and post processors help to convert the source records. Conversion can be performed through the TYPO3 Extbase property mapper or directly through the database.
