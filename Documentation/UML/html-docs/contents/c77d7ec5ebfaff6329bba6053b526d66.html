register

UMLOperation
t3import_export :: T3importExport :: Service :: DataBaseConnectionService :: register

Description

Registers a database.

Parameters

Direction Name Type Description
in identifier string

Unique identifier for the connection to be registered. If another database is already registered under this identifier, the current will not be registered. To be used in TypoScript configuration of data sources and targets.
in hostName string

Host name or IP address of the connection.
in databaseName string

Name of the default database to use with this connection.
in userName string

A user name to authenticate with for this connection.
in password string

A password to authenticate with.
in port int

Port to use for this connection. Default: 3306
